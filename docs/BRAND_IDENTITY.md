# Identidad visual de Menuly

## Activos oficiales

- Logo horizontal: `/brand/menuly-logo-horizontal.png`
- Símbolo: `/brand/menuly-mark.png`
- Fuente original: `Menuly_Manual_Identidad.docx`

El logo horizontal se usa en navegación, acceso, documentos y soportes comerciales. El símbolo queda reservado para favicon, icono PWA y espacios reducidos. No se deforma, recolorea ni recibe sombras, contornos o efectos.

## Paleta

- Navy principal: `#0C1F30`
- Coral de acento: `#F0643A`
- Marfil de fondo: `#FBF8F3`
- Grafito secundario: `#374151`

Los botones principales usan navy para mantener contraste accesible. El coral identifica acentos, enlaces, foco y detalles de marca. Verde, ámbar y rojo se reservan para estados funcionales.

## Tipografía

- Titulares: serif editorial, con Georgia como alternativa web disponible.
- Interfaz y cuerpo: Inter, Arial y sans-serif del sistema.
- Botones: sans-serif semibold o bold.

## Superficies

- Fondo general marfil.
- Tarjetas blancas con bordes suaves y sombras discretas.
- Sin neón, brillos, efectos 3D ni colores saturados ajenos a la paleta.
